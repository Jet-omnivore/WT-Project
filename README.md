# MediRemind 💊 — Smart Medicine & Health Appointment Tracker

A web application to help users track their daily medicines and doctor appointments.

## Tech Stack
- React.js + Vite
- Tailwind CSS
- React Router DOM

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Pages
| Route | Page | Description |
|---|---|---|
| `/` | Home | Landing page with hero, features, testimonials |
| `/login` | Login | Login form with validation |
| `/register` | Register | Signup form with role selector |
| `/dashboard` | Dashboard | Medicine tracking + appointments overview |
| `/medicines` | Medicines | Medicine list with search and add modal |
| `/appointments` | Appointments | Appointment list with add modal |
| `/about` | About Us | Team info and tech stack |

## Color Scheme
- Primary: `#0D7377` (Dark Teal)
- Secondary: `#14A085` (Medium Teal)
